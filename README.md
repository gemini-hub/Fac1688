# 1688 工厂判定助手

Edge MV3 扩展：点击扩展后读取当前 1688 页面公司名称，通过百度爱企查查询经营范围，并按关键词判断：

- 经营范围包含“生产”或“制造”：工厂
- 经营范围未包含上述关键词：贸易

## 安装调试

1. 打开 Edge/Chrome，进入 `edge://extensions/`。
2. 打开“开发人员模式”。
3. 点击“加载解压缩的扩展”。
4. 选择本目录：`\1688 插件`。
5. 打开任意 1688 公司页或商品页，点击扩展图标查看结果。

## 注意

爱企查可能出现验证码、登录或反爬限制。扩展会创建一个不激活的临时标签页打开爱企查搜索结果，找到公司名称对应的详情页后进入详情页，再从公司简介/经营范围中解析关键词；如果未解析到经营范围，会按规则暂判定为贸易，并提供爱企查详情页或搜索链接用于人工复核。
